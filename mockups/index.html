Sift

Daily Reflection Tool - UI Mockups

Core User Flows

  • 1. Dashboard / Home
    Overview showing current streak, trends, and recent entries
  • 2. New Entry
    Write a daily reflection with option to Save or Refine
  • 3. AI Refinement Conversation
    Chat-style interface where AI asks clarifying questions
  • 4. Summary & Scoring
    Review polished summary and set wellbeing score (1-10)
  • 5. Entry Detail View
    Read a complete saved entry with full narrative and context

Additional States

  • 6. Crisis Detection
    Example of resource banner shown during crisis language detection
  • 7. Empty State
    Dashboard when user has no entries yet
Note: These are static HTML/CSS mockups demonstrating the UI design and user flows. No JavaScript is used. Each page represents a different state in the application.
